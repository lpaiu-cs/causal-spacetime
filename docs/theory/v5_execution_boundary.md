# V5 Execution Boundary

Milestone 46 does not execute v5. It exports a planned-only v5 preregistration
spec and verifies that no v5 production outputs exist.

Milestone 47 may execute v5 only from the M46 preregistered spec. Milestone 48
should apply fixed carry-forward criteria to v5 outputs if Milestone 47
produces a diagnostic-complete v5 bundle.

Stress tests remain impossible unless a later fixed-criteria evaluation
produces a carry_forward or provisional family.

Milestone 46 does not regenerate v4 manifests, rerun v4 fits, rerun
carry-forward evaluation, retune thresholds, run stress tests, or infer metric
geometry.
