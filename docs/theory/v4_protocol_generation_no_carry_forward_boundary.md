# V4 Protocol Generation No-Carry-Forward Boundary

Milestone 44 is an execution milestone, not a decision milestone. It may
generate v4 manifests and v4 diagnostic outputs from the M43 preregistration.

M44 must not write v4 carry-forward decisions. It must not write a v4
carry-forward registry. It must not write a v4 stress-test handoff plan. It
does not run stress tests and does not infer metric geometry.

M45 is the future fixed-criteria decision layer. Only that later milestone may
apply the fixed M34 carry-forward criteria to the v4 diagnostic-complete output
bundle.
